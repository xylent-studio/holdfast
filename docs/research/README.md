# Research Archive

This folder holds synthesized research snapshots, review programs, and other evidence that should inform Holdfast without silently rewriting the product contract.

## Operating rule

Research docs are:

- evidence
- hypotheses
- input for prioritization

Research docs are not:

- the product contract
- automatic roadmap law
- a replacement for the control docs

If research conflicts with [docs/control](/C:/dev/GitHub/Holdfast/docs/control), surface the conflict explicitly and resolve it on purpose.
